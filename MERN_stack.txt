Full Stack MERN (set up project):
mkdir myNewProject
cd myNewProject
touch server.js
npm init -y
npm install express cors mongoose


in server.js:
const express = require('express');
const app = express();
const port = 8000;
    
app.listen(port, () => console.log(`Listening on port: ${port}`) );

After that React create a new prject:
npx create-react-app your-project-name-here

install in the new react project:
npm install @reach/router
npm install axios
npm install cors

run project from client folder:
npm run start

run project from main folder:
nodemon server.js


Add extention to Chrome
